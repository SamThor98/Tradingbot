    • Live trading: enabling requires risk acknowledgment plus typing ENABLE; pause blocks new approvals; 2FA protects sensitive actions; automation stays off until you opt in.

    Kronos
    
          
    Forecast the next moves
    
          
    
            Kronos is an open-source foundation model for financial candlesticks. Enter a ticker to
            project the next sessions of price action — advisory only, never an order.
          
    
        
    
        
    
          Run a forecast
          How it works
        
    
      
    

      
    
        
    
          
    Kronos forecast
    
          Projected candles, direction, and confidence for a single symbol.
        
    

        
    
          
    
            
    
              
              
            
    
            
    
              
              
            
    
            
    
              
              
            
    
            
    
              
            
    
          
    
          
    
            Shows the median path, a p10-p90 range cone, and P(up) vs a flat baseline. Intraday (5m/15m)
            is Schwab-sourced (~10 trading days). Base model on CPU can take up to ~90s.
          
    
        
    

        
    

        
    
          
    Enter a ticker and run a forecast to see history with the projected path.
    
        
    

        
    
      
    

      
    
        
    
          
    How Kronos works
    
          Context for interpreting the forecast responsibly.
        
    
        
      
          
    • 
            Inputs. Recent daily OHLCV candles (the same data feeding the scanner) are
            tokenized and fed to a decoder-only transformer pre-trained on global K-line sequences.
          
      • 
          
    • 
            Output. A projected candle path, a net direction, an expected percentage move
            over the horizon, and a confidence proxy derived from the predicted path.
          
      • 
          
    • 
            Shadow mode. In the scanner, forecasts attach context without changing
            ranking or placing orders.
          
      • 
          
    • 
            Degrades safely. If the inference service is offline, the workspace shows a
            clear status message instead of failing.
